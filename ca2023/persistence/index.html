Persistence



Enumeración


En la descripción del reto se nos dice que antes al hacer una petición por GET hacia /flag se conseguia un poder inmenso y sabiduria, aunque ahora solo devuelve datos aleatorios, pero que la leyenda dice que funciona una vez de cada 1000 intentos

Podemos probarlo, al hacer una petición con el comando curl hacia la ruta/flag, el servidor nos devuelve algo aleatorio en este caso un texto sin demasiado sentido

❯ curl http://188.166.174.127:30475/flag  
|d{hE,U2lLo;B!6g<?g91_l

¿Tal vez es algo cifrado? pues no, al hacer otra petición devuelve un texto diferente

❯ curl http://188.166.174.127:30475/flag  
_x,c5p'42supoX8xTa/8G


Explotación


Se dice que funciona 1 de cada 1000 veces, bueno hagamos 1000 peticiones para que una de ellas devuelva algo interesante, iniciamos con un script importando log de la libreria pwn, además de la libreria la requests y sys, y obligamos a que se nos proporcione el host y puerto en el primer argumento como host:port

#!/usr/bin/python3
from pwn import log
import requests, sys

if len(sys.argv) < 2:
    log.failure(f"Uso: python3 {sys.argv[0]} <host:port>")  
    sys.exit(1)

A esto le agregamos una barra que estaremos actualizando y definimos el target

bar = log.progress("")

target = f"http://{sys.argv[1]}/flag"

Ahora haremos 1000 peticiones guardandola en una variable request y por cada una de ellas actualizaremos la barra mostrando el numero de peticiones enviadas

for num in range(0, 1000):
    bar.status(f"Requests: {str(num)}")  
    request = requests.get(target)

Cuando la cadena HTB este en la respuesta significa que nos esta mostrando la flag asi que la mostramos actualizando la barra y salimos del script

if "HTB" in request.text:  
    bar.success(f"Flag: {request.text}")
    sys.exit(0)

Nuestro script final de fuerza bruta nos quedaria de la siguiente manera

#!/usr/bin/python3
from pwn import log
import requests, sys

if len(sys.argv) < 2:
    log.failure(f"Uso: python3 {sys.argv[0]} <host:port>")  
    sys.exit(1)

bar = log.progress("")

target = f"http://{sys.argv[1]}/flag"

for num in range(0, 1000):
    bar.status(f"Requests: {str(num)}")
    request = requests.get(target)

    if "HTB" in request.text:
        bar.success(f"Flag: {request.text}")
        sys.exit(0)

Al ejecutar el script con host:port como primer argumento este empezara a relizar peticiones hasta llegar 1000 peticiones totales o hasta que encuentre la flag

❯ python3 persistence.py 188.166.174.127:30475  
[█] Requests: 10

Despues de un rato encuentra la flag y se nos muestra, hemos completado el reto

❯ python3 persistence.py 188.166.174.127:30475
[+] Flag: HTB{y0u_h4v3_p0w3rfuL_sCr1pt1ng_ab1lit13S!}
