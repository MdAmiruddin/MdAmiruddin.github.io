Explotación


Enviamos 1 y bueno... ¿conseguimos la flag?, en fin esta es la solución el reto ;)

>> 1
HTB{g3t_r34dy_f0r_s0m3_pwn}

Vamos a crear un script para no hacerlo tan pequeño, iniciamos recibiendo host:port como el primer argumento y creando una conexión a le máquina

#!/usr/bin/python3
from pwn import remote, log
import sys, re

if len(sys.argv) < 2:
    log.failure(f"Uso: python3 {sys.argv[0]} <host:port>")  
    sys.exit(1)

host, port = sys.argv[1].split(":")
shell = remote(host, port)

Enviamos un 1 para recibir la flag y recibimos toda la data, en ella jugando con expresiones regulares buscamos el patron de la flag y lo mostramos por pantalla

shell.sendline(b"1")

output = shell.recvall()
match = re.search(b"HTB\{.*\}", output)
log.success(f"Flag: {match.group(0).decode()}")

Finalmente nuestro script queda asi, al ejecutarlo con host:port como argumento hace todo el trabajo por nosotros ademas de buscar y obtener la flag

#!/usr/bin/python3
from pwn import remote, log
import sys, re

if len(sys.argv) < 2:
    log.failure(f"Uso: python3 {sys.argv[0]} <host:port>")  
    sys.exit(1)

host, port = sys.argv[1].split(":")
shell = remote(host, port)

shell.sendline(b"1")

output = shell.recvall()
match = re.search(b"HTB\{.*\}", output)
log.success(f"Flag: {match.group(0).decode()}")
