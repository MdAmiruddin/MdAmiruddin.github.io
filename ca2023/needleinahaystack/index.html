Needle in a Haystack



Enumeración


Una vez descomprimido el zip solo nos deja un compilado llamado haystack

❯ ls
 haystack

Al ejecutarlo nos pide un input, le pasamos una string como test y al enviarla nos devuelve una cadena realmente sin ningun tipo de sentido

❯ ./haystack
Hit enter to select a recording: test  
..."hloo"


Explotación


Al probar listar cadenas de caracteres imprimibles con strings nos muestra varias, si grepeamos por HTB ya que asi empieza la flag, podemos verla directamente

❯ strings haystack | grep HTB
HTB{d1v1ng_1nt0_th3_d4tab4nk5}
